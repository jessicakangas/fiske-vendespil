# Fiske Vendespil 

Et simpelt memory-spil lavet i HTML, CSS og JavaScript.

## Sådan spiller du
- Vend kort og find par
- Du har 80 sekunder
- Find alle par før tiden løber ud!

## Funktioner
- Vendespil med 16 kort
- Timer (80 sekunder)
- Lyd ved klik på krabbe
- Taleboble med beskeder
- Animation (krabbe bevæger sig)

## Teknologier
- HTML5
- CSS3 (inklusiv animationer)
- JavaScript (DOMmanipulation)
